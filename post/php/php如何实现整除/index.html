php如何实现整除

以前在用C/C++的时候，整除就是’/‘，还要特别注意这个问题，今天用到php调试的时候出错了，最后发现：在php中’/‘除法是默认带有小数部分的，php自带有几个函数可以对小数进行处理

1. round()函数，四舍五入

<?php echo round(7/3); //2 ?>

2. ceil()函数，进一法

也就是取比该数大的最小整数

<?php echo ceil(7/3); //3 ?>

3. floor()函数，舍一法

也就是取比该数小的最大整数

<?php echo floor(7/3); //2?>
