一个普通 Go 开发的三年

注：本文所有示例代码都可以在 blog-code 仓库中找到

先回答一下最近的热门问题，没有断更，还会更新，只是最近换城市来回倒腾，然后休息了几天，争取会在 Q3 把之前的 Go 进阶训练营 笔记系列更新完

时间还是过的比较快，回顾一下还没怎么感觉就已经毕业三年了，三年前的今天(2021/07/13) 正是我入职的日子，所以我也水一篇总结吧

独自一人在他乡，很多不经意的瞬间还是会感觉到一些孤独，先感谢家属鑫姐的陪伴和支持，过多的话就不再文章里面虐🐶了，今天主要聊一下工作和学习

为什么我选择用 Go 语言？

这应该是我在大二大三的时候的一个选择，当时我主要是用 PHP/JS 来写网站，从大一开始自学，用了一两年之后 CURD 已经写的非常熟练了，这时候就感觉有一些瓶颈，然后想一直写脚本语言也不是个办法（动态一时爽，一直动态一直爽）所以就萌生了学习一门编译型的语言。

当时写 C 感觉太底层了，学 C++ 又觉得有点难，用 JAVA 又感觉有点啰嗦，当时不知道怎么的知道了 Go，然后一看

  • 有 Google 背书，一时半会儿应该黄不了
  • 有 Docker 这样大型成熟的开源产品（刚开始都不知道 K8s）
  • 语法非常简单，关键词很少只有 20 多个，原生支持协程（这个应该是当时最大的动力）

纠结了一下，然后就他了，现在回想起来当时的我还是太年轻，没有太考虑市场环境，当然如果考虑了，可能就会投入 JAVA 的怀抱了

我还是很幸运的，因为我上班的第一家公司就用上了 Go 大学两三年也没白学

在深圳不加班是一种什么体验？

说到工作，我也非常的幸运，从实习到第一份正式工作都没有怎么加过班，这其中包括了华为，大疆这些久负盛名的公司

我是秋招接到 DJI 的 offer 然后去了如故团队实习，然后实习完刚回学校没多久，团队就解散了，虽然呆的不久，但是还是挺喜欢当时整个的团队氛围，大家都很热情和融洽。随后正式入职之后就加入了我之前 Leader 的团队做 K8s 开发

从技术来讲，就我个人而言，最大的几个收获我觉得是

  • 有比较大的自主性，参与并且主导了我们一些系统的设计，虽然由于一些经验原因，其中有些涉及做的挺烂的，但是纸上得来终觉浅,绝知此事要躬行，没有掉过坑，走在路上就不会那么小心
  • 有比较充分的个人时间，说人话就是不怎么加班，我基本上除了项目刚开始很忙的时候，大部分时间都可以做到准点下班，有时间充实自己，虽然前面才说了纸上得来终觉浅，但是如果不在巨人肩膀上，做出来的东西很有可能就是井底之蛙了（PS: 合适很重要，也不要好高骛远）

从团队上来讲

  • 领导都很 Nice，坤哥是我刚入职时候的导师，从实习开始就一直在他的团队，感谢他对我一直以来的看中和照顾，不知道他会不会看这篇文章，但是我觉得他是一个很好的领导，属于那种温和型的，但是也会努力给团队争取资源，虽然有时候也会有一些上面管理上的压力。
  • 整体大家的氛围还是很不错的，虽然很多时候我们的想法和方向上可能会有一些差别，不过不过技术没有对错，虽然我已经撤了但是还是坚持我的观点哈哈😄

这一趴先到这里，祝福大家工作顺利越来越棒，最重要的是要天天开心呀~

工作之后如何学习？

学习是一个比较大的话题，这里就简单的聊一聊

不知道大家有没有这种感受，工作之后就不怎么看书了，每天就在上班，下班躺着，之间循环，周末也不太想动，能躺着我为什么要坐起来？

不总结不反思，不学习是不会进步的，或者说会相对慢一点

  • 输出倒逼输入，费曼学习法还是比较出名了，只有能够教会别人我们才能更好的掌握，输出的过程中本身也是对我们知识的一次重新梳理，写文章只是其中的一种方式，更好的应该是面对面的讲授
  • 看一些经典书籍和 Paper，虽然技术日新月异，概念层出不穷，但是地基并没有那么多新东西，这些都是可以很长一段时间受益的
  • 关注一些技术会议分享，在这儿可以看到很多公司的实践，不一定要大公司，往往很多相同体量公司的实践会更有价值（PS: 警惕推广和过度美化）
  • 关注一些最新的顶会论文，可以帮我们了解很当前的当下学术界的前沿

看到这儿了不关注一下公众号: mohuishou，是不是说不过去？

写博客真的有人看吗？

有人看，我博客其实搭建了很久了现在还可以看到第一篇文章在 2015 年，之前断断续续的也有更新，但是都比较少并且不成体系，从去年 8 月份开始，我就开始写一些体系一点的笔记，更新频率也上来了，去年 8 月到现在更新了将近 80 篇文章，比过去 5 年加起来还多

看 GA 的数据，从去年的 7 月底开始，网站的月活用户开始稳步上升

今年用阿里云日志服务做了一下数据统计，可以看到现在每个月的月活实际已经破万，并且相对比较稳定，虽然因为换工作我已经将近一个月没更新了

从 3 月初在博大的建议下开始搞微信订阅号到现在也有 1000+ 的粉丝了，算是完成了上半年的 KPI 的吧，感谢博大，煎架等各位大佬的微信群流量😄

都看到这儿了加个关注呗，下一个三年继续加油~
